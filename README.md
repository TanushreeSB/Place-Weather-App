# 🌦️ Place Weather App

A Java-based desktop application that provides real-time weather information for any location using the OpenWeatherMap API. Built with Java Swing, the application offers a simple and interactive interface for users to fetch and view current weather conditions.

---

## 🚀 Features

- 🌍 Search weather by city/place name  
- ⏱️ Fetch real-time weather data using OpenWeatherMap API  
- 🖥️ User-friendly GUI built with Java Swing and AWT  
- 🔗 API integration using `HttpURLConnection`  
- ⚠️ Error handling for invalid inputs and API failures  
- 🔑 Secure API key configuration (user-defined)

---

## 🛠️ Tech Stack

- **Language:** Java  
- **UI Framework:** Swing, AWT  
- **API Integration:** OpenWeatherMap API  
- **Networking:** HttpURLConnection  
- **Data Format:** JSON  

---

## ⚙️ How It Works

1. User enters a city name in the application.  
2. The app sends a request to the OpenWeatherMap API.  
3. Weather data is retrieved in JSON format.  
4. The application parses and displays relevant information (e.g., temperature, conditions) in the UI.  

---

## 🎯 Objectives

- Provide accurate, real-time weather updates  
- Demonstrate integration with external APIs  
- Implement reliable data fetching using Java networking  
- Build an interactive desktop UI using Swing  
- Ensure secure handling of API keys  

---

## 🔐 API Key Setup

To run this application:

1. Get your API key from OpenWeatherMap  
2. Replace the placeholder API key in the code with your own  
3. Run the application  

## 💡 Summary

This project demonstrates core Java development concepts including GUI design, API integration, and data handling. It serves as a foundation for building more advanced, scalable weather applications.
